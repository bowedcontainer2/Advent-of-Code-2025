# Advent of Code 2025

Just for fun! Solving Advent of Code problems in Python.

## About

This repository contains my solutions to the Advent of Code 2025 challenges. Each day's solution is implemented in Python.

## Structure

- `day1.py` - Day 1 solution
- `day1Input.txt` - Day 1 input file

## Usage

Run each day's solution with:
```bash
python3 day1.py
```
